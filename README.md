# Gonzalo Sangrá — Portafolio 2026

Portafolio profesional desarrollado con React, Vite, Framer Motion y Tailwind CSS. Este proyecto destaca mi trabajo en branding, diseño 3D, producción audiovisual y desarrollo web potenciado por Inteligencia Artificial.

## ✨ Características

- **Diseño Premium**: Interfaz moderna con estética "glassmorphism" y animaciones fluidas.
- **Carruseles Dinámicos**: Visualización interactiva de múltiples assets por proyecto.
- **Lightbox**: Galería a pantalla completa para el análisis detallado de imágenes.
- **Hero Animado**: Fondo con formas geométricas cinéticas en colores neón.
- **Optimización de IA**: Implementación estratégica utilizando tecnologías de Google DeepMind (Gemini & Antigravity).

## 🚀 Tecnologías

- **React 18**
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## 🛠️ Instalación y Uso

1. Clonar el repositorio.
2. Instalar dependencias: `npm install`
3. Ejecutar en modo desarrollo: `npm run dev`
4. Generar build de producción: `npm run build`
